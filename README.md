# 🚀 About

**ESP32-Horcrux** is a massive collection of ESP32-based projects, tools, networking utilities, IoT applications, embedded experiments, automation examples, and educational implementations.

Instead of maintaining dozens of separate repositories, this project centralizes everything into one well-organized toolkit for developers, students, makers, and embedded enthusiasts.

Whether you're learning ESP32 or building advanced IoT systems, **ESP32-Horcrux** aims to be your all-in-one development repository.

# ⚙ Getting Started

Clone the repository

```bash
git clone https://github.com/aayush-xid-su/Esp32-Horcrux.git
```

Open the desired project.

Install required libraries.

Select your ESP32 board.

Upload the sketch.

Open Serial Monitor.

Done!
